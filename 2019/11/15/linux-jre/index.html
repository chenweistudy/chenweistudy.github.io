    linux安装jre

    linux linux安装jre
    创建时间:2019-11-15 17:20
    阅读:
    1. 1.首先将jre解压到随便目录 比如/usr/lib/jre1212
    2. 2.vim /etc/profile
    3. 3.如果不小心修改错了PATH值，可以使用
    4. 4.然后 java -version查看jre版本
    5. 1.rpm –ivh jdk-6u20-linux-i586.rpm
    6. 2.vim /etc/profile
    7. 3. 使用 . /etc/profile生效

    #方式一

    1.首先将jre解压到随便目录 比如/usr/lib/jre1212

    2.vim /etc/profile

    加上:
export JAVA_HOME=/usr/lib/jre1212/jre1.7.0_72

export CLASSPATH=$JAVA_HOME/lib

export PATH=$PATH:$JAVA_HOME/bin (这里相当于是在原有path后追加$JAVA_HOME/bin，使用echo $PATH可查看值)

然后 . /etc/profile生效

    3.如果不小心修改错了PATH值，可以使用

    export PATH=...来重新设置

    4.然后 java -version查看jre版本

    #方式二 安装rpm包

    1.rpm –ivh jdk-6u20-linux-i586.rpm

    之后jdk会默认安装在/usr/java里面

    2.vim /etc/profile

    加上:
export JAVA_HOME=/usr/java/jdk
export CLASSPATH=$JAVA_HOME/lib
export PATH=$PATH:$JAVA_HOME/bin (这里相当于是在原有path后追加$JAVA_HOME/bin，使用echo $PATH可查看值)

    3. 使用 . /etc/profile生效
