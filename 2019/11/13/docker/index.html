    3.docker查看镜像信息

    docker inspect tomcat （如tomcat）
    配置镜像加速器
    https://blog.csdn.net/yimenglin/article/details/93718784

    1.登录：https://dev.aliyun.com/search.html

    2.进入阿里云账号获取加速器地址

    修改daemon配置文件/etc/docker/daemon.json来使用加速器

    sudo mkdir -p /etc/docker

    sudo tee /etc/docker/daemon.json <<-‘EOF’

    {
    “registry-mirrors”: [“https://czrunef9.mirror.aliyuncs.com"]
    }

    EOF

    sudo systemctl daemon-reload

    sudo systemctl restart docker

    4.docker操作容器内部目录

    1.查看 容器ID 
    docker ps -a
2.通过容器ID查看内部
    docker exec -it id /bin/bash
3.退出容器
    exit

    5.docker复制文件

    1.从主机复制到容器    
    docker cp 主机路径 容器ID:容器内部路径
2.从容器复制到主机
    docker cp 容器ID:容器内部路径 主机路径   
3.启动nginx镜像，没有会自动pull
    docker run -d --name mynginx nginx  
4.停止一个容器，根据容器 id 进行删除
    docker stop id   
5.删除一个容器
    docker rm id   
6.进入一个正在运行的容器
    docker attach id
